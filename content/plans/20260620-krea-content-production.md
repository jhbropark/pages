# bbbb.beauty × Krea 콘텐츠 제작 계획

> Science to Message, Beauty to Experience. 기술은 정확하게, 메시지는 쉽게.

Krea를 bbbb.beauty의 메디컬·바이오 비주얼 제작 파이프라인에 통합해, 시각
방향 탐색부터 프리비주얼라이제이션, 채널 자산 생성까지의 속도를 높이되
의학적 정확성과 브랜드 컬러 시스템은 기존 검수 체계로 그대로 보장한다.

이 문서는 `strategy.json`의 비주얼 시스템·컴플라이언스·채널 규칙과
`CONTENT_PLAYBOOK.md`의 제작 순서를 전제로 한다.

## 1. Krea를 쓰는 이유와 범위

- **목적:** 3D/MOA·다큐멘터리·교육 콘텐츠 본 제작에 들어가기 전, 컨셉과
  무드를 빠르게 탐색하고 소셜(Instagram·LinkedIn)용 비주얼을 효율적으로
  생산한다.
- **포지션:** Krea는 컨셉 탐색·프리비주얼·소셜 비주얼 **도구**다. 고객
  납품용 최종 MOA의 과학적 고증을 대체하지 않는다(§5 참조).
- **불변 기준:** Deep Navy 60 / Soft Beige 30 / Aqua Blue 10 비율, 시네마틱
  매크로 라이팅, 하이퍼 리얼리스틱 분자·세포·피부 장벽이라는
  `visual_system`의 선호 톤을 모든 결과물이 따른다.

## 2. Krea 기능 ↔ 제작 단계 매핑

| 제작 단계 | Krea 기능 | 산출물 |
|---|---|---|
| 컨셉·무드 탐색 | Realtime / Image | 시각 방향 후보 보드 |
| 브랜드 톤 고정 | Train(커스텀 스타일) | bbbb.beauty 전용 스타일 모델 |
| 정지 비주얼 | Image + Edit(인페인트) | Instagram 정사각형, LinkedIn 가로 카드 |
| 모션·MOA 프리비주얼 | Image-to-Video | 세포 이동·성분 전달 6~10초 시퀀스 |
| 품질 업스케일 | Enhancer / Upscale | 게시·납품 해상도 마스터 |

## 3. 브랜드 비주얼 일관성: 커스텀 스타일 학습

매번 프롬프트로 컬러·톤을 재현하면 편차가 크다. Krea Train으로 브랜드
전용 스타일을 한 번 학습해 일관성을 확보한다.

1. 학습 데이터: 기존 `images/` 중 컬러 비율과 라이팅이 합격한 컷
   (예: `images/linkedin/diverse-series-v2/`, `images/channel-six-types/`)을
   큐레이션한다. 저품질·왜곡 컷은 제외한다.
2. 학습 목표: Deep Navy 배경, Aqua Blue 포인트, Soft Beige 여백, 시네마틱
   매크로 심도라는 시그니처를 재현하는 스타일.
3. 검증: 학습 후 동일 프롬프트로 10컷을 생성해 `visual_color_ratios`
   기준(합계 100, 60/30/10 근사)과 `avoid` 항목 위반 여부를 점검한다.
4. 버전 관리: 스타일 모델은 `v1`, `v2`로 명명하고 채택 버전을 이 문서에
   기록한다.

## 4. 콘텐츠 기둥별 Krea 워크플로우

`pillar_cycle_20_posts` 순서를 따르되, 기둥별로 Krea 활용 포인트를 정한다.

- **science_communication(25%)** — 추상 개념(번역·이해의 벽)을 은유하는
  Image 비주얼. 텍스트 헤드라인은 기존 `generate_content.py` 템플릿으로
  올려 가독성과 컬러 비율을 통제한다.
- **moa_craft(20%)** — Image-to-Video로 성분 전달·작용 기전 프리비주얼을
  만들어 Instagram 릴스/스토리 후보로 사용한다. 구조 정확성은 §5 검수.
- **industry_solution(20%)** — 더마·제약·바이오·의료기기별 시나리오를
  Realtime로 빠르게 분기 탐색해 업종 맞춤 비주얼을 제안한다.
- **methodology(15%)** — "고증 → 프리비주얼 → 본 제작" 워크플로우 자체를
  비하인드 카드로 시각화한다.
- **portfolio(10%)** — 공개 승인된 자산만 Upscale로 마스터링해 쇼케이스.
- **philosophy(10%)** — 절제된 단일 오브젝트·여백 중심 비주얼.

## 5. 컴플라이언스 가드레일 (필수)

`strategy.json`의 `compliance`와 `visual_system.avoid`를 그대로 적용한다.

- AI 생성 비주얼은 **컨셉·마케팅·프리비주얼용**이다. 분자 구조, 세포 형태,
  작용 기전의 과학적 정확성은 **메디컬 고증 자문위원 검수**를 통과해야
  한다. 검수 전 자산은 MOA "사실"로 게시하지 않는다.
- 의학적 구조를 왜곡하는 장식적 시각화, 장난감 같은 저품질 3D는 폐기한다.
- 고객사·전환율·임상 성과는 공개 승인 출처가 없으면 생성하지 않는다.
  가상 컷은 `예시 상황`·`검토 프레임`으로 명시한다.
- "국내 최초", "세계 유일" 등 최상급과 금지 의학 표현을 캡션·이미지
  텍스트에 넣지 않는다.

## 6. 파일럿 배치 (첫 주 제작 큐)

브랜드 스타일 모델(§3) 채택 후, 다음 4개 컨셉을 파일럿으로 제작한다.
각 컨셉은 LinkedIn 한국어→영어 쌍과 Instagram 카드를 함께 만든다.

1. **성분 전달의 가시화** (moa_craft) — Image-to-Video 6초, DM `MOA`
2. **차갑지 않은 과학** (science_communication) — 따뜻한 랩 무드 Image, DM `LAB`
3. **하나의 세포, 하나의 변화** (moa_craft) — 극단적 심도 시네마틱, DM `FILM`
4. **업종별 한 장면** (industry_solution) — 더마/제약 분기 비주얼, DM `UX`

각 캡션·해시태그는 `CONTENT_PLAYBOOK.md`의 CTA 3종 규칙과
`validate_generated_content` 기준(해시태그 개수·`#` 접두사 포함)을 만족해야
한다.

## 7. 파이프라인 통합

- Krea 산출물은 `images/`의 기존 디렉터리 규칙을 따라 저장한다
  (예: `images/krea/<batch>/<concept>-ko.jpg`).
- LinkedIn 가로 자산은 정사각형 재사용 없이 `1200×627(1.91:1)`로 별도
  생성·업스케일한다.
- 헤드라인 텍스트는 가급적 Krea 배경 위에 기존 렌더러로 올려 폰트·컬러
  비율·로고 위치를 통제한다.
- 최종 컷은 PR에서 CMO·브랜드 에디터 승인을 받는다.

## 8. 운영·일정

| 단계 | 담당 | 산출물 |
|---|---|---|
| 스타일 모델 학습·검증 | 비주얼 | v1 스타일 + 검증 10컷 |
| 파일럿 4컨셉 제작 | 비주얼 | LinkedIn 쌍 + IG 카드 |
| 고증·컬러·컴플라이언스 검수 | 메디컬 자문·에디터 | 합격/수정 노트 |
| 채널 큐 등록 | 운영 | `queue` / `linkedin/queue` 반영 |

발행 시간은 기존 캘린더(LinkedIn 09:00·15:00, Instagram 12:30·19:00 KST)를
따른다.

## 9. 성공 기준

- **정성:** 브랜드 컬러 비율·라이팅 일관성, 메디컬 고증 합격률.
- **정량:** `strategy.json`의 KPI(인바운드 DM·문의, CTR, 저장·공유)와 함께,
  컨셉 1건당 제작 리드타임 단축 정도를 별도로 기록한다.

## 10. 미해결·확인 필요

- Krea 유료 플랜·상업적 사용 라이선스 범위와 학습 데이터 사용 약관 확인.
- 학습에 사용할 자산의 NDA·클라이언트 RA 공개 승인 범위 확인.
- 채택할 영상 모델 라인업(Krea가 제공하는 image-to-video 모델 중)과
  컷당 비용·렌더 시간 벤치마크.
